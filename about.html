Metahandle


What is Metahandle?

Metahandle is a platform to interact with the BSV blockchain. It allows to create and crawl so called Metahandles. These are BSV transactions whose OP_Return output contains a word chosen by the user and some more information. Their basic use case is to reference other transactions containing content with a human readable name. Beyond this there are many further use cases of Metahandles.

Terms of Service

Metahandle.net is just an interface between the users and the blockchain. It is not responsible for the availability of the BSV blockchain or the integrity of the data.
It is not allowed to use Metahandle to reference to transactions containing illicit content.

Privacy Protection

Metahandle.net does not store, process or share any private data and doesn't want to change this. Period.
If it ever happens that Metahandle.net starts processing private data the privacy protection explanation will be updated.
